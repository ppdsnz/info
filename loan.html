PPDSNZ INTER LIBRARY LOAN

Introduction

Provides inter library loan borrowings service for a printed materials, seminar proceedings, report or a copy of original replacement resources which are not available in the Library. These items will be acquired for temporary loan from other academic library institutions in Malaysia. The purpose is to facilitate and support teaching, learning and research in the University.

Which patron category is eligible?

All category students, lecturers and staffs can used this service.

How to request?

Fill up the Inter Library Loan request form which can be obtained at the main counter and Library Portal : http://ilmusea.umt.edu.my/web/guest/forms Item request will be processed within five (5) working days. Item will be in hand within two (2) weeks. Items will be loaned for month duration. Borrowing period can be extended subject to item availability from the borrowers' institution. No charges are incurred.
